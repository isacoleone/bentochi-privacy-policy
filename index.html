Política de Privacidade

Última atualização: 02 de julho de 2026

A sua privacidade é importante. O Bentochi foi desenvolvido para ajudá-lo a organizar seus conteúdos de forma simples e segura.

Coleta de informações

O Bentochi não coleta, transmite ou armazena dados pessoais em servidores externos.

Todos os dados criados pelo usuário, como arquivos, links, anotações e projetos, permanecem armazenados localmente no seu dispositivo.

Arquivos

Quando você adiciona um arquivo ao Bentochi, o aplicativo apenas mantém uma referência para que ele possa ser aberto posteriormente. O conteúdo do arquivo não é enviado para nenhum servidor.

Internet

O Bentochi não utiliza servidores próprios nem realiza sincronização de dados pela internet.

Compartilhamento de dados

Nenhuma informação é compartilhada, vendida ou disponibilizada para terceiros.

Segurança

Como os dados permanecem no seu dispositivo, a segurança das informações depende das proteções oferecidas pelo próprio macOS, incluindo permissões de acesso aos arquivos.

Alterações nesta política

Esta Política de Privacidade poderá ser atualizada futuramente para refletir mudanças no aplicativo. A versão mais recente estará sempre disponível nesta página.
